# Essencial do HTML

### Nesse módulo é visto o basico de HTML, como:

- tags: head, body e html
- boilerplate
- tag de parágrafo p
- tags de cabeçalho H1 até H6
- tags de listas: listas ordenadas (ol) e não ordenadas (ul)
- tag de link: a
- tag de imagem: img
